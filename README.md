# CasseBrique-HTML

Petit jeu de casse-brique en HTML5 Canvas, CSS et JavaScript natif.

## Apercu

Ce projet propose une version simple et jouable d'un casse-brique:
- Deplacement de la raquette au clavier
- Balle avec collisions murs/raquette/briques
- Grille de briques colorees
- Message de victoire quand toutes les briques sont detruites

## Lancer le jeu

1. Ouvrir le fichier `casse brique.html` dans un navigateur web.

Option recommandee avec VS Code:
- Utiliser une extension de serveur local (ex: Live Server) pour lancer la page.

## Contenu du projet

- `casse brique.html`: page principale contenant le jeu
- `assets/css`: dossier CSS (reserve pour evolutions)
- `assets/js`: dossier JavaScript (reserve pour evolutions)
- `assets/img`: dossier images (reserve pour evolutions)
